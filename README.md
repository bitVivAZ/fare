Fare is todo application prototype I made using ReactJS. 

### Tech Stack 
- HTML
- CSS
- JavaScript
- ReactJS
- Ant Design
- Heroku
- NodeJS
- MongoDB

### Features
- Responsive Design
- Add, Edit and Delete Todocards
- Add, Edit and Delete Subtasks
